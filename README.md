Projeto: Landing Page - Logix Chaves MX S
Este projeto é uma página de produto (landing page) desenvolvida como parte dos meus estudos em desenvolvimento front-end. O objetivo aqui foi criar uma interface profissional e atraente para apresentar um teclado premium, focando em design, organização e usabilidade.

🎯 Objetivo do Projeto
Este não é um site de vendas real, mas sim um projeto de portfólio criado para praticar:

A estruturação de páginas utilizando Vite.

A criação de interfaces modernas e focadas na experiência do usuário.

A implementação de uma identidade visual clara para um produto.

🛠️ Tecnologias
Framework: Vite

Linguagem: TypeScript

Foco: Desenvolvimento de interfaces e prototipagem de produtos.
